# Life Calendar (React)

## Setup & Run

1. Open your terminal.
2. Navigate to the project directory:
   ```bash
   cd "/Users/kundansingh/Desktop/LIfe Calendar"
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
